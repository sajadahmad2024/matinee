### Loki Deployment (`loki-deployment.yml`)

This deployment sets up Grafana Loki for log aggregation in the `monitoring` namespace.

### **Key Configuration Details:**

- **Node Affinity**:
    - Scheduled only on nodes with label: `node-role=monitoring`.
- **Tolerations**:
    - Accepts nodes with taint: `monitoring-only=true:NoSchedule`.
- **Volumes**:
    - Uses `loki-data-pvc` for storing log chunks.
- **Retention**:
    - Configured with a **15-day log retention**.

### **Allowed Configuration Changes**

- You **may only change** the following section in `local-config.yaml` if required:

```yaml
limits_config:
  retention_period: 15d
```

- ❌ Do **not change** other configurations like ports, storage backend, schema, or ruler settings.
